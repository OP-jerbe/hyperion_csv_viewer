Quick Start Guide

This application allows users to load, view, and analyze time-series CSV data files using an interactive Plotly-based interface.

Getting Started

  1. Open the application. The main window will launch.
  2. Click the "Load CSV Files" button to select one or more CSV files from the file browser.
  3. After selection, the application will display the selected CSV files in the canvas at the bottom of the window and the dropdown boxes will be populated with the headers of the data columns in CSV file (excluding Time).
  4. Select up to four data columns from the dropdown boxes to plot.
  5. Select the x-axis variable from the dropdown box. Time is default.
  6. Click either "Plot Data" to generate an interactive Plotly graph or, from the menu bar, select "Save" > "Save Plot as HTML" to save an HTML file of the interactive Plotly graph
Tip: Give your plot a meaningful title before saving it.

Plotly Graph Features

The interactive Plotly graph provides powerful tools for exploring your data:

  • Zoom by dragging: Click and drag a box to zoom into a specific region of the plot.
  • Pan mode: Use the toolbar or hold Shift and drag the graph to pan around.
  • Zoom in/out: Use the toolbar buttons to zoom in and out.
  • Reset axes: Click the home icon in the top-right to reset the view to default.
  • Manually enter axis limits: Click on any axis limit to type in a custom range.
  • Scaling axis limits: Click and drag any axis limit to change the scaling.
  • Pan Individual Axes: Click and drag from the center of any axis to pan that axis without affecting other axes.
  • Toggle traces: Click trace names in the legend to hide/show them dynamically. Double-click a trace to hide all other traces.
Tip: You can also save the plot as a PNG file by using the toolbar menu.

Troubleshooting

  • If no data appears, double-check that your CSV file contains a Time column and that the datetime format is MM/DD/YYYY hh:mm:ss AM/PM.
  • Only CSV files with standard column formatting are supported.
  • CSV files should be loaded in chronological order. Check the Selected csv files canvas to ensure the files are in the correct order.
  • Ensure that the CSV files are not too large or that too many CSV files are loaded at once. The application may struggle with very large datasets.
  • Reach out to Joshua Erbe (j.erbe@oregon-physics.com) if you experience loading or rendering issues.
